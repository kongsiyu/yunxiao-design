## 关键技术细节

- `auth` 配置存储在 `~/.yunxiao/config.json`
- 格式：`{ pat, userId, userName, orgId, orgName }`
- 支持环境变量（优先）和配置文件（fallback）两种认证方式
- API 基础 URL：`https://openapi-rdc.aliyuncs.com`
- Node.js ESM 模块（`"type": "module"`）
- shebang 行导致 `node --check` 无法用于语法检查，改用 `node src/index.js --version`

## 注意事项

- PowerShell 中不能用 `&&`，用 `;` 代替
- PowerShell 中复杂引号用 `-F <file>` 传入 gh 命令
- GitHub 仓库分支是 `master` 不是 `main`，CI 配置要对应
- node_modules 不应该被提交（需要检查 .gitignore）
