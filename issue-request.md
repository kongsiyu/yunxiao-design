# 需求：优化云效CLI环境变量配置流程

## 背景
当前云效CLI需要手动设置环境变量进行配置，用户体验不够友好。希望能像`gh auth`命令一样，提供交互式配置界面。

## 需求描述
实现一个交互式的配置命令，允许用户：
1. 输入个人访问令牌
2. 自动查询用户所属的组织列表
3. 提供选择界面让用户选择默认组织
4. 保存配置到本地环境变量或配置文件

## 技术要求
- 使用云效OpenAPI查询组织列表
- API端点：`https://{domain}/oapi/v1/platform/organizations`
- 请求头需要`x-yunxiao-token`（个人访问令牌）
- 返回参数包含组织ID、名称等信息
- 支持分页查询

## API文档参考
- 服务接入点：需要替换{domain}
- 需要个人访问令牌
- 组织ID用于后续API调用
- 权限：组织只读权限

## 预期功能
- `yunxiao config` 或类似的命令
- 交互式输入个人访问令牌
- 自动获取并展示组织列表
- 用户选择默认组织
- 保存配置到本地
